# Goodime Frontend (Flutter App)

Goodime Frontend is a cross-platform mobile application built using Flutter.  
It is part of the Goodime AI ecosystem and connects to a Node.js + Express backend powered by MongoDB and AI services.

---

## Features

- Fast and responsive Flutter UI
- Connected to live backend API
- AI-powered search and responses
- Displays dynamic data from MongoDB
- Real-time API integration
- Works on Android (APK) and Web

---

## Tech Stack

- Flutter (Dart)
- HTTP package (API calls)
- Provider / State Management (if used)
- Node.js Backend (separate repo)
- MongoDB Atlas (database)
