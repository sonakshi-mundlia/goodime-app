# Goodime AI Backend

Goodime AI Backend is a scalable REST API built using **Node.js, Express.js, and MongoDB Atlas**.  
It powers a Flutter-based AI application that handles data processing, search, and intelligent responses.

---

## Features

- User authentication (JWT-based if implemented)
- RESTful APIs for Flutter frontend
- MongoDB Atlas cloud database integration
- Fast and scalable backend architecture
- AI-ready structure for future enhancements

---

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Dotenv
